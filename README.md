# Charites - 美化工坊 ✨

![Image](https://raw.githubusercontent.com/SenriYuki/SillyTavern-Charites/refs/heads/main/CharitesLogo.png)

**SillyTavern 可视化主题定制插件** — 不懂代码也能轻松 DIY 属于自己的酒馆美化。

> Charites（卡律忒斯）— 希腊神话中掌管美丽与优雅的三女神。

## 功能一览

### 🎨 配色与气泡
- HSL 调色盘生成整套配色方案，一键出主题
- USER / CHAR 气泡独立设置：颜色、透明度、渐变（主色 + 配色 + 角度）
- 气泡分离排列：宽度、对齐方式、左右边距、圆角、间距
- CHAR 可一键跟随 USER 渐变，也可完全独立

### 🖼️ 头像与布局
- 头像位置（默认 / 顶部居左 / 顶部居中）、大小、形状、边框圆角
- 头像装饰框 URL
- 名字跟随头像位置，发送时间独立定位（名字下方 / 置右 / XY 座标）

### ✏️ 名字样式
- USER / CHAR 独立字体、字号、渐变色
- 文字特效：外描边（16方向均匀模拟）、阴影、发光

### 🔧 顶栏 / 底栏 / 图标
- 顶部栏、图标容器、底部输入栏背景色可视化调色（色盘 + 透明度滑条）
- 顶部图标大小、颜色、透明度
- 支持自定义图标 URL 替换（含 hover 效果图）
- 兼容旧版图标系统与新版 Navbar

### 🌨️ 特效
- 飘落特效：雪花 / 花瓣 / 星星，可调数量、大小、速度、颜色
- 飘落层级：全局（最前方）或 背景（不遮挡气泡）
- 鼠标特效：拖尾 / 涟漪 / 光晕
- 特效可独立导入 / 导出分享

### 📦 主题管理
- 导出为 ST 原生格式或 Charites 格式
- 本地保存 / 加载多套主题方案

### 🧩 其他
- 加载画面自定义（背景色、背景图、加载图标）
- 聊天区背景图、气泡顶部装饰图
- 强调色边框、自定义 CSS 注入
- 代码片段快捷开关（去按钮阴影、输入框透明、隐藏顶栏阴影等）

## 安装

1. 打开 SillyTavern → 扩展 → 安装扩展
2. 输入此仓库地址，点击安装
3. 刷新页面，顶栏出现 🎨 图标即安装成功

## 使用

点击顶栏的 🎨 图标打开美化工坊面板，所有设置实时预览，关闭预览即恢复原样。

**快速上手：**
- 拖动「色相 / 饱和度 / 明度」滑条即可生成一套配色
- 导出时选择「导出为ST格式」可分享给不用本插件的人

## 兼容性

- SillyTavern 1.12.0以上（含新版 Navbar）
- 桌面端 & 移动端浏览器
