# Charites — Мастерская стилей ✨

![Image](https://raw.githubusercontent.com/SenriYuki/SillyTavern-Charites/refs/heads/main/CharitesLogo.png)

**Плагин для визуальной настройки темы SillyTavern** — DIY стилей для вашей таверны, даже если вы не разбираетесь в коде.

> Charites (Хариты) — три богини красоты и изящества в греческой мифологии.

## Обзор функций

### 🎨 Цветовые схемы и облачка сообщений
*   Генерация готовой цветовой схемы с помощью ползунков HSL (тон/насыщенность/яркость) — создание темы в один клик.
*   Независимая настройка облачков USER и CHAR: цвет, прозрачность, градиент (основной цвет + дополнительный цвет + угол).
*   Настройка расположения облачков: ширина, выравнивание, отступы слева и справа, скругление углов, расстояние между сообщениями.
*   Облачка CHAR могут одним кликом наследовать градиент USER или быть полностью независимыми.

### 🖼️ Аватар и компоновка
*   Положение аватара (по умолчанию / сверху слева / сверху по центру), размер, форма, скругление рамки.
*   URL для декоративной рамки аватара.
*   Имя пользователя следует за положением аватара, время отправки позиционируется отдельно (под именем / справа / координаты X Y).

### ✏️ Стиль имени
*   Независимые шрифты, размер и градиентный цвет для USER и CHAR.
*   Эффекты для текста: внешняя обводка (равномерная имитация по 16 направлениям), тень, свечение.

### 🔧 Верхняя панель / Нижняя панель / Иконки
*   Визуальная настройка цвета фона верхней панели, контейнера иконок и нижней панели ввода (цветовой круг + ползунок прозрачности).
*   Размер, цвет и прозрачность иконок на верхней панели.
*   Поддержка замены иконок на произвольный URL-адрес изображения (включая изображение для эффекта при наведении).
*   Совместимость со старой системой иконок и новой панелью навигации (Navbar).

### 🌨️ Эффекты
*   Эффект падающих объектов: снежинки / лепестки / звезды. Настройка количества, размера, скорости и цвета.
*   Слой эффектов: поверх всего (поверх облачков) или на фоне (не перекрывает облачка).
*   Эффекты курсора: шлейф / рябь / свечение.
*   Возможность независимого импорта/экспорта эффектов для обмена.

### 📦 Управление темами
*   Экспорт в нативный формат ST или в формат Charites.
*   Сохранение и загрузка нескольких наборов тем локально.

### 🧩 Прочее
*   Настройка экрана загрузки (цвет фона, фоновое изображение, иконка загрузки).
*   Фоновое изображение области чата, декоративное изображение над облачками.
*   Акцентная рамка, инъекция произвольного CSS.
*   Быстрое включение/отключение фрагментов кода (убрать тень кнопок, прозрачное поле ввода, скрыть тень верхней панели и т.д.).

## Установка

1.  Откройте SillyTavern → Расширения (Extensions) → Установить расширение (Install Extension).
2.  Вставьте адрес этого репозитория и нажмите «Установить».
3.  Обновите страницу. Если на верхней панели появилась иконка 🎨, установка прошла успешно.

## Использование

Нажмите на иконку 🎨 на верхней панели, чтобы открыть панель Мастерской стилей. Все настройки применяются в реальном времени, а после закрытия панели возвращаются к исходному виду.

**Быстрый старт:**
*   Перетаскивайте ползунки «Тон / Насыщенность / Яркость», чтобы сгенерировать цветовую схему.
*   При экспорте выберите «Экспорт в формат ST», чтобы поделиться темой с теми, у кого не установлен этот плагин.

## Совместимость

*   SillyTavern версии 1.12.0 и выше (включая новую панель навигации Navbar).
*   Компьютерная и мобильная версии.
